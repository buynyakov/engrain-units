[Assessment Demo] Simple plugin to display list of units.

== Description ==

Simple plugin to retrive units from API and display them via [shortcode_unit_list] shortcode.


== Installation ==

1. Upload `engrain-units` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `[shortcode_unit_list]` in your post or page where you want the list of units displayed.
